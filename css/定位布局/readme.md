# 定位
1. 静态定位
 position:static;
 - 默认值，元素按照正常的文档流进行定位

 2. 相对定位
position：relative
- 相对定位的元素，相对于其原来的文档流位置进行定位
- 不会脱离文档流
 3. 绝对定位
position: absolute
- 需要一个参考标准，这个参考标准是拥有定位属性的父元素，父元素都没定位属性则参考body
- 绝对定位的元素会脱离文档流
- 水平垂直居中

4. 固定定位
position：fixed
- 相对浏览器窗口定位
- 即使滚动页面元素也保持不动
- 脱离文档流

5. 粘性定位
position：sticky
